# Bank Management System in Java

## Overview

A console-based Bank Management System developed using Java and OOP concepts.

## Features

* Login Authentication
* Create Account
* Deposit Money
* Withdraw Money
* Check Balance
* View Account Details

## Technologies Used

* Java
* OOP
* ArrayList
* Exception Handling

## How to Run

Compile:
javac *.java

Run:
java Main
